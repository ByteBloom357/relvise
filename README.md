Relvise — Finance and Consultancy Landing Page

A modern, fully responsive, and pixel-perfect landing page for a financial and consultancy agency named Relvise. Built from scratch using semantic HTML5, CSS3, and the BEM (Block, Element, Modifier) methodology for clean and scalable styles.
🚀 Features

    Fully Responsive Design: Seamlessly adapts to desktops, tablets, and mobile devices (tested down to 320px screens).

    BEM Methodology: Highly structured and clean CSS architecture, preventing class collisions and deeply nested selectors.

    Modern Layouts: Developed using Flexbox for clean alignment, flexible components, and grid-like columns.

    Inline Background Images (_ibg): Uses a smart utility approach to turn standard image tags into high-performance cover backgrounds via CSS.

    Interactive Elements: Features custom styled buttons, interactive inputs, and smooth hover effects on links and interactive cards.

🛠️ Tech Stack

    Markup: HTML5 (Semantic elements like <header>, <main>, <section>, <article>, <footer>)

    Styling: CSS3 (Flexbox, Media Queries, Custom CSS reset, calc() functions)

    Fonts: Google Fonts (Montserrat)

🔧 Getting Started

To view the project locally, you don't need any complex build steps. Simply follow these instructions:

    Clone the repository:
    Bash

    git clone https://github.com/ByteBloom357/relvise-landing.git

    Navigate to the project folder:
    Bash

    cd relvise-landing

    Open the project:

        Double-click index.html to open it directly in your browser.

        Alternatively, use the Live Server extension in VS Code for a real-time preview.

📐 Optimization & Best Practices Applied

    Box-Sizing: box-sizing: border-box and content-box used dynamically where padding adjustments are essential.

    Performance: Google Fonts loaded with &display=swap to prevent invisible text during rendering.

    Text Truncation: Applied text-overflow: ellipsis along with white-space: nowrap for specific item boundaries to ensure UI stability even with dynamic text.
